### Άσκηση 2
Ας υποθέσουμε ότι:
* Χ είναι ο μήνας των γενεθλίων σας (Χ=1..12)
* Υ είναι η μέρα των γενεθλίων σας (Υ=1..31)
Υπολογίστε το Ζ=Χ\*Υ
Πως είναι το Ζ στο δυαδικό; Γράξτε τι υπολογισμούς κάνατε. 
Για παράδειγμα: Αν Χ=12, Υ=31, τότε Ζ=372. To 372 στο δυαδικό είναι: 101110100.

### Άσκηση 3 
Ας υποθέσουμε ότι έχουμε τον εξής 
```
1Χ0Υ1
```
Όπου:
* Το Χ είναι 0 αν ο μήνας που γεννηθήκατε είναι μονός (περιττός) αλλιώς είναι 0
* Το Υ είναι 0 αν η μέρα που γεννηθήκατε είναι μονή (περιττή) αλλιώς είναι 1

Πως είναι δεκαδικό σύστημα ο αριθμός που σχηματίστηκε;

### Άσκηση 5
Η χιλιομετρική οδική απόσταση μεταξύ Αθήνας και Θεσσαλονίκης είναι 500 χιλιόμετρα ([πηγή](https://vriskoapostasi.gr/el/%CE%B4%CE%B9%CE%B1%CE%B4%CF%81%CE%BF%CE%BC%CE%AE/%CE%91%CE%B8%CE%AE%CE%BD%CE%B1/%CE%98%CE%B5%CF%83%CF%83%CE%B1%CE%BB%CE%BF%CE%BD%CE%AF%CE%BA%CE%B7)). Ας υποθέσουμε ότι χρησιμοποιούμε 3 bits για να καθορίσουμε ένα σημείο της διαδρομής. Επίσης έστω ότι η αρίθμηση των χιλιομέτρων ξεκινάει από την Αθήνα και σταματάει στη Θεσσαλονίκη. Ένα αυτοκίνητο χαλάει στο σημείο:
```
ΧΥΖ
```
Όπου:
* Το Χ είναι 0 αν ο μήνας που γεννηθήκατε είναι μονός (περιττός) αλλιώς είναι 0
* Το Υ είναι 0 αν η μέρα που γεννηθήκατε είναι μονή (περιττή) αλλιώς είναι 1
* Το Ζ είναι 0 αν το επίθετό σας ξεκινάει από φωνήεν αλλιώς είναι 1

Από ποιο έως ποιο σημείο της διαδρομής (σε χιλιόμετρα) βρίσκεται το αυτοκίνητο;
