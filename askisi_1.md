Σε αυτή τη σειρά θα πρέπει να υλοποιήσετε στο δικό σας υπολογιστή μερικές από τις δραστηριότητες που κάναμε στις τρεις πρώτες διαλέξεις. Θα πρέπει να παραδώσετε **ένα** αρχείο μορφοποιημένου κειμένου. Το αρχείο αυτό θα πρέπει να το φτιάξετε σε ένα από τα παρακάτω προγράμματα επεξεργασίας κειμένου: LibreOffice, Microsoft Word, Google Docs. Κάθε άσκηση θα πρέπει να βρίσκεται σε μία δική της επικεφαλίδα με όνομα στυλ "Ηeading 2". Κάθε screenshot θα πρέπει να έχει μήκος 10 cm (width) και θα πρέπει το ratio να είναι σταθερό. Δηλαδή μπορεί το ύψος (height) να είναι διαφορετικό έτσι ώστε να διατηρείται το width στα 10 εκατοστά χωρίς να γίνεται distort η εικόνα. Κάθε screenshot θα πρέπει να έχει caption ένα απλό κείμενο όπως αυτό: "η μνήμη τους υπολογιστή μου".

### Άσκηση 2
Ας υποθέσουμε ότι:
* Χ είναι ο μήνας των γενεθλίων σας (Χ=1..12)
* Υ είναι η μέρα των γενεθλίων σας (Υ=1..31)
Υπολογίστε το Ζ=Χ\*Υ
Πως είναι το Ζ στο δυαδικό; Γράξτε τι υπολογισμούς κάνατε. 
Για παράδειγμα: Αν Χ=12, Υ=31, τότε Ζ=372. To 372 στο δυαδικό είναι: 101110100.

Όπου:
* Το Χ είναι 0 αν ο μήνας που γεννηθήκατε είναι μονός (περιττός) αλλιώς είναι 0
* Το Υ είναι 0 αν η μέρα που γεννηθήκατε είναι μονή (περιττή) αλλιώς είναι 1

Πως είναι δεκαδικό σύστημα ο αριθμός που σχηματίστηκε;

Ένα αυτοκίνητο χαλάει στο σημείο:
```
ΧΥΖ
```
Όπου:
* Το Χ είναι 0 αν ο μήνας που γεννηθήκατε είναι μονός (περιττός) αλλιώς είναι 0
* Το Υ είναι 0 αν η μέρα που γεννηθήκατε είναι μονή (περιττή) αλλιώς είναι 1
* Το Ζ είναι 0 αν το επίθετό σας ξεκινάει από φωνήεν αλλιώς είναι 1

Από ποιο έως ποιο σημείο της διαδρομής (σε χιλιόμετρα) βρίσκεται το αυτοκίνητο;
